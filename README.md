### Retail-customer-insights-sales-optimization
📊 Project Overview
    This project analyzes retail customer shopping behavior using 3,900 transaction records to uncover insights that support sales optimization and data-driven decision-making.

---

## 🛠 Tech Stack
    Python (Pandas)
    MySQL
    Power BI

---

## 🔎 Key Analysis
    Revenue by gender and age group
    Subscribers vs non-subscribers spending
    Top-performing and highest-rated products
    Impact of discounts on sales
    Shipping type comparison
    Customer segmentation 

---

## 📈 Business Impact
    The analysis provides actionable insights to:
    Improve customer retention
    Optimize discount strategies
    Increase subscription growth
    Enhance overall sales performance
