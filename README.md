# 📊 Retail Customer Behavior & Sales Analytics

## 1️⃣ Project Overview

This project analyzes customer shopping behavior using transactional data from **3,900 purchases** across multiple product categories.

The objective is to uncover insights into:

- Customer spending patterns  
- Product preferences  
- Subscription behavior  
- Discount usage  
- Customer segmentation  

The analysis supports **data-driven strategic decisions and sales optimization**.

## 3️⃣ Exploratory Data Analysis (Python)

Data cleaning and preprocessing were performed using **Python (Pandas & NumPy)**.

### 🔹 Data Preparation Steps

**Data Loading**
- Imported dataset using `pandas`.

**Initial Exploration**
- Used `df.info()` to inspect structure.
- Used `df.describe()` for summary statistics.

**Missing Data Handling**
- Identified 37 missing values in `review_rating`.
- Imputed missing values using the **median rating of each product category**.

**Column Standardization**
- Renamed columns to `snake_case` for consistency and readability.

**Feature Engineering**
- Created `age_group` column by binning customer ages.
- Created `purchase_frequency_days` from purchase history.

**Data Consistency Check**
- Verified redundancy between `discount_applied` and `promo_code_used`.
- Dropped `promo_code_used` after validation.

**Database Integration**
- Connected Python to **PostgreSQL**.
- Loaded cleaned DataFrame into the database for SQL-based analysis.

## 4️⃣ Business Analysis Using SQL (PostgreSQL)

Structured SQL queries were written to answer key business questions:

1. **Revenue by Gender**  
   Compared total revenue generated by male vs. female customers.

2. **High-Spending Discount Users**  
   Identified customers who used discounts but still spent above average.

3. **Top 5 Products by Rating**  
   Ranked products based on highest average review ratings.

4. **Shipping Type Comparison**  
   Compared average purchase amounts between Standard and Express shipping.

5. **Subscribers vs. Non-Subscribers**  
   Compared total revenue and average spend across subscription status.

6. **Discount-Dependent Products**  
   Identified five products with the highest percentage of discounted purchases.

7. **Customer Segmentation**  
   Classified customers into:
   - New  
   - Returning  
   - Loyal  

8. **Top 3 Products per Category**  
   Listed the most purchased products within each category.

9. **Repeat Buyers & Subscription Behavior**  
   Analyzed whether customers with more than five purchases are more likely to subscribe.

10. **Revenue by Age Group**  
    Calculated total revenue contribution for each age group.
